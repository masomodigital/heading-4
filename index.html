Learning HTML: Understanding Heading 4 (<h4>)

Introduction

Welcome to the lesson on <h4>! This heading is smaller than <h3> and is often used for sub-sections in your webpage content. Using headings helps organize your information.

Explanation

The <h4> tag defines the fourth level of headings in HTML. It's useful for further dividing sections under <h3> headings. You can place text between the <h4> and </h4> tags to create a heading.

Example

Letâ€™s see an example in code:

<!DOCTYPE html>
<html>
<head>
    <title>Subheading Level 4 Example</title>
</head>
<body>
    <h4>My Favorite Colors</h4>
</body>
</html>

When you run this code, you'll see "My Favorite Colors" appear as a subheading on the page.
